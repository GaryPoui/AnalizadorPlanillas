📋 Columnas del Template (Plantilla_Precios_Compras)

Fila 0: CUIT del proveedor Fila 1: Encabezados de columnas Fila 2+: Datos extraídos Columnas: Cód. Artículo → Código del artículo del proveedor Descripción artículo → Descripción principal Desc. adicional artículo → Especificaciones técnicas Sinónimo → Código alternativo Cód. Lista → Código de lista de precios Desc. Lista → Nombre de la lista Moneda → ARS / USD / EUR Unidad → Un / m / kg / caja Precio → Numérico sin símbolo Bonif. → % de descuento Fecha vigencia desde → DD/MM/YYYY Fecha vigencia hasta → DD/MM/YYYY
